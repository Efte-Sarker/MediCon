# MediCon

MediCon is a next-generation mobile healthcare platform built to bridge the gap between patients, healthcare professionals, and intelligent medical systems. The application provides an integrated, bilingual experience designed to make medical assistance seamless, safe, and accessible.

## Project Scope

The goal of MediCon is to provide comprehensive digital healthcare management through:

- **AI-Powered Symptom Triage**: Helps patients quickly assess their symptoms and recommends appropriate action (home care vs. urgent care).
- **Telemedicine & Consultations**: Supports booking remote or in-person doctor appointments, featuring built-in video-call infrastructure and dynamic queues.
- **Intelligent Prescriptions & Medicine Interactions**: Digital prescription management with an automated drug-interaction checker that flags potential conflicts between medications.
- **Emergency SOS Protocols**: A dedicated one-tap emergency system providing immediate access to critical protocols, nearby hospitals, and localized emergency contacts.
- **Health Records & Vitals Tracking**: Comprehensive dashboard for monitoring patient vitals and securely storing digital medical reports.
- **Bilingual Support**: Fully localized in English and Bengali for wider accessibility.

## Tech Stack

- **Framework**: React Native with [Expo](https://expo.dev/) (Expo Router for navigation)
- **Language**: TypeScript for end-to-end type safety
- **State Management**: Zustand
- **Styling**: Custom Theme System (Vanilla RN Stylesheets)
- **Localization**: `i18next` for seamless English/Bengali switching
- **Animations**: `react-native-reanimated` for performant 60fps micro-animations

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npx expo start` to launch the development server

*Note: This project is under active development. Ensure you have the proper Expo environments configured before running.*
